Installation Guide

Follow these step-by-step instructions to set up the Boutique Hotel & Spa theme on your local machine or hosting provider.

1. Local Setup

To run the website locally on your machine for development or testing:

  • Extract the downloaded zip file into your preferred workspace.
  • Open the extracted folder in a code editor like VS Code.
  • Run a local development server. If using VS Code, install the "Live Server" extension and click "Go Live".
  • Alternatively, use Python's built-in HTTP server:
> python -m http.server 8000

Navigate to http://localhost:8000 in your web browser.

2. Hosting Deployment

Deploying to a live server is straightforward for this static HTML/CSS/JS template.

  • FTP/cPanel: Connect to your web hosting via FTP and upload the entire contents of the folder directly into the public_html or www directory.
  • Netlify/Vercel/GitHub Pages: Simply drag and drop the project folder into your Netlify/Vercel dashboard, or connect your GitHub repository for automated deployments.
  • Ensure the index.html file remains in the root directory.

3. Folder Structure

Understanding the directory layout is essential for modifying the template.

/boutique-hotel-spa ├── css/ // Core Stylesheets (global.css, index.css) ├── js/ // Core Javascript files (global.js, components.js) ├── images/ // Web-optimized image assets ├── fonts/ // Local font files (if any) ├── index.html // Landing Page (Home 1) ├── about.html // About Us Page ├── services.html // Spa & Hotel Services Catalog └── ... // Other HTML templates

Customization Guide

Learn how to easily modify colors, typography, layout, and content to match your brand identity.

Colors (CSS Variables)

All core colors are centrally managed using CSS variables in the css/global.css file. Modify these directly to change the theme across all pages.

/* In css/global.css root scope */ :root { --primary-color: #000000; /* Update for main background */ --accent-color: #cda87a; /* Update for gold buttons/links */ --text-color: #ffffff; /* Update for primary text */ }

Fonts Selection

The template utilizes Google Fonts for premium typography.

  • Open the <head> tags in any HTML file to locate the Google Fonts stylesheet link.
  • To change the font family entirely, navigate to css/global.css and update the font-family properties inside the body and heading tags.

Content Replacement

Content areas are logically divided and heavily commented in the HTML documents. This allows you to easily identify sections.

<!-- ================= HERO SECTION ================= --> <section class="hero"> <h1>Welcome to Paradise</h1> <p>Experience tranquility...</p> </section>

Note: Replace images in the /images folder, ensuring new files have similar aspect ratios to maintain layout structural integrity.

Page Structure

Breakdown of the included page templates to assist in navigation and modifications.

Public Facing Pages

  • Home (index.html): The main entry point featuring the Hero slider, introductory snippets, featured rooms, and a spa summary.
  • About Us (about.html): Covers hotel history, core philosophy, team details, and timeline.
  • Services (services.html): A grid-based layout detailing Spa treatments, Dining, and Hotel amenities.
  • Contact (contact.html): Features an interactive map integration snippet and a functional-ready inquiry form.

Authentication & Dashboards

  • Login/Register (login.html, register.html): Optimized forms with fluid responsiveness and theme adaptability.
  • Dashboard (dashboard-profile.html): Profile management, loyalty points viewer.
  • Past Stays (dash-past-stays.html): Component for reviewing booking history and invoices.
